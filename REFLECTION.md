# inventory-hub
Coursera assignment for Full Stack Integration


1. Copilot assisted in generating code by adding a typed model and a cached API wrapper. It updated the Blazor product page with safe loading/error states and JSON deserialization methods. Using Copilot I created a ProductService that houses the code to fetch data from the back end and display the data on the web page if there is data to display.
2. The biggest challenge I encountered was encapsulating the functionality of the front end and back end communication into a ProductService class. Copilot optimized my code and made it more readable and also added an if statement to handle null cases.
3. I learned that Copilot is an effective tool at generating, optimizing, and securing code as well as suggesting edits in a full-stack development context.
